ht-python-pyramid
==================

Sample project that use Python Pyramid for Heroku training.

How to deploy
--------------

### Step1  

Clone app from Github  

```
$ git clone git@github.com:flect/ht-python-pyramid.git
$ cd ht-python-pyramid/
```

### Step2  

Create Heroku application

```
$ heroku create
```

### Step3  

Deploy application
```
$ git push heroku master
```

### Step 4

Visit application

```
$ heroku open
```
